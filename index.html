¿Que es Git-Hub?

GitHub es una plataforma de desarrollo colaborativo basada en la nube que utiliza Git como sistema de control de
versiones. Permite a los desarrolladores almacenar, administrar, compartir y colaborar en proyectos de software
de manera eficiente.


Para que sirve?

GitHub permite gestionar proyectos de software mediante control de versiones, facilitando la colaboración entre
desarrolladores, el almacenamiento remoto de código y la automatización de procesos como pruebas y despliegues.
Además, ofrece herramientas para la organización del trabajo, como Issues y Pull Requests, permitiendo un
desarrollo más eficiente y ordenado.


Comandos comunes:

  • Crear o clonar un repositorio

    • git init: Inicializa un nuevo repositorio en la carpeta actual.
    git clone URL_DEL_REPO: Clona un repositorio remoto en tu computadora.

  • Enviar cambios a GitHub

    • git push origin main: Sube los cambios al repositorio remoto en la rama main.

  • Obtener cambios desde GitHub

    • git pull origin main: Descarga y fusiona los cambios más recientes del repositorio remoto.

  • Crear y trabajar con ramas

    • git branch nombre_rama: Crea una nueva rama.
    git checkout nombre_rama: Cambia a la rama especificada.
    git merge nombre_rama: Fusiona una rama con la actual.

  • Gestionar cambios y errores

    • git status: Muestra el estado actual del repositorio.
    git log: Muestra el historial de commits.
    git reset --hard HEAD~1: Revierte al estado del commit anterior.
